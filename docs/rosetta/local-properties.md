# Local Properties — Merged into XStore

> **C5 (local mapping properties) has been merged into [xstore-crossdb.md](xstore-crossdb.md).**
>
> Local properties are tightly coupled to XStore — they exist only to provide
> internal join keys for cross-store associations. Without XStore, local properties
> are dead code (filtered out of execution plans, invisible to user queries).
>
> See [XStore, Local Properties, Source/Target IDs & Cross-DB](xstore-crossdb.md) for the full analysis.
